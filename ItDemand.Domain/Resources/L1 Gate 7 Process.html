All demand that is in the status Gate 7 Closeout Checklist can be progressed to the status Completed once the initiative has been completed.

  • Regional PMO or the Project Manager (PM) can request Corporate PMO to set any Demand to Complete.
  • A request to Corporate PMO to set the demand to Complete can be via email, Instant Message, or by assigning the closeout to a PMO member and submitting it.
  • No review (SOX, Medical Validation, Data Protection, etc.) will take place for Proceed Locally demand by Corporate PMO as this reposnibility falls to the Regions if required.
